Tortoise GIS
============

.. toctree::
   :maxdepth: 2
   :caption: Contents:

Geospatial integration for Tortoise ORM.
This package enables Tortoise ORM to utilize geospatial features and functionalities,
integrating GIS databases to the ORM.

This project **IS NOT** production ready. The roadmap for it is described below.

   * [V] Add Geometry support.
   * [X] Add tests.
   * [X] Add support for SpatiaLite and MySQL.
   * [X] Add Geography support.
